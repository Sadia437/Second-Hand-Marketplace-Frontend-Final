🛍️ Secondhand Marketplace - Frontend

A modern, full-featured marketplace application for buying and selling pre-loved items. Built with React 18, Tailwind CSS, and React Query for a premium user experience.

✨ Key UI Enhancements

🪄 Custom Floating Particles: High-performance, pure CSS-based animated backgrounds on Home and Products pages for a modern look.

🔠 Smart Capitalization: Automated text formatting for category titles and collections (e.g., "Electronics" instead of "ELECTRONICS").

💎 Glassmorphism Design: Premium UI using backdrop-blur, custom shadow-optimized cards, and smooth transitions.

🌟 Features
For Buyers

🔍 Advanced Category Search - Quickly find items in Electronics, Furniture, Vehicles, and Property.

🛍️ Interactive Product Cards - View condition, location, and verified seller badges at a glance.

♻️ Sustainable Shopping - Join the circular economy by purchasing quality pre-loved items.

📱 Fully Responsive - Optimized for seamless browsing on Mobile, Tablet, and Desktop.

For Sellers

🏪 Seller Dashboard - Effortlessly manage products and track order statuses.

✅ Verified Badges - Built-in trust system for verified sellers and trusted partners.

💰 Revenue Insights - Keep track of sales and active listings.

🚀 Tech Stack
Frontend Core

React 18 - Component-based UI library.

Tailwind CSS 3 - Utility-first CSS framework for rapid styling.

React Query - Powerful data fetching and state synchronization.

React Router v6 - Client-side routing for seamless navigation.

Libraries & Tools

DaisyUI - Accessible Tailwind CSS components.

Axios - Promise-based HTTP client for API requests.

React Hot Toast - Lightweight notifications for user actions.

📦 Installation & Setup
# Clone the repository
git clone https://github.com/Sadia437/Second-Hand-Marketplace-Frontend-Final.git

# Navigate to project directory
cd Second-Hand-Marketplace-Frontend-Final

# Install dependencies
npm install

# Run development server
npm run dev
